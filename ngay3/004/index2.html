thuộc tính display trong css
        - khái niệm kiểu block ( thẻ div là thẻ điển hình )
        có 2 đặc điểm : luôn xuất hiện trên 1 hàng mới , chiếm 100% độ rộng có sẵn trên màn hình
        - khái niệm kiểu inline ( thẻ span là thẻ điển hình )
        có 2 đặc điểm : luôn nằm trền cùng 1 hàng , độ rộng chiếm tương ứng nội dung bên trong
        RESET CSS : trình duyệt firefox , chrome có css mặc định
        xóa bỏ đoạn css mặc định thì dùng reset css

        có khoảng 100 thẻ html chia làm 2 nhóm
        -display có tác dụng đảo kiểu block sang inline
